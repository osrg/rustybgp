# RFC Compliance

RustyBGP implements and is compliant with the following RFCs:

- [RFC 7911 - Advertisement of Multiple Paths in BGP (Add-Path)](https://www.rfc-editor.org/rfc/rfc7911.html)
- [RFC 8950 - IPv4 Network Layer Reachability Information (NLRI) with an IPv6 Next Hop](https://www.rfc-editor.org/rfc/rfc8950.html)
